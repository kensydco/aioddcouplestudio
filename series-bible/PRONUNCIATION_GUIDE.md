# PRONUNCIATION_GUIDE.md — TTS & Naming

> Ensures HeyGen voices say things correctly, and standardizes Gladys's running "misnaming" gag so it's consistent across episodes. Check this file during scripting and QA.

---

## 1. How to write for clean TTS

- Spell tricky terms phonetically in a **director note**, not inside spoken dialogue (stage directions never go in spoken lines — see scriptwriting prompt).
- Prefer spaces/hyphens over camelCase in scripts so TTS reads them naturally ("Chat G P T", not "ChatGPT") **for Milo's correct usage**.
- Avoid raw URLs, emoji, and symbols in spoken lines; spell out "dot com" only if essential (usually avoid).
- Keep numbers readable: "twenty twenty-six," not "2026," if a year must be spoken.

## 3. Gladys's approved misnamings (the gag)

Gladys **always** gets the name wrong — affectionately, consistently. Rotate from this approved list so it feels like a running bit, not a typo. **Milo (or the takeaway) should make the correct name clear**, so the audience still learns it.

| Real tool | Gladys calls it |
|-----------|-----------------|
| ChatGPT | "ChatterBox GPT," "the Chatty-Box," "that GPT fella" |
| AI (general) | "the Artificial," "this A.I. business," "the smart machine" |
| Google / search | "The Google Machine," "the Googly," "the search box" |
| chatbots | "the talking boxes," "the answer machines" |
| the internet | "the internet machine," "the World Wide Web-thing" |
| social media | "FaceSpace," "the Insta-gram," "the TikTalk" |
| image generators | "the picture printer," "the magic crayon" |
| algorithm | "the al-GORE-ithm" (mock-formal), "the recipe thingy" |
| app | "the application doohickey," "the little square" |
| smartphone | "the pocket telephone" |

**Rules for the gag:**
- One or two misnamings per episode — don't bury the lesson.
- The **correct** name must be said or shown at least once (usually by Milo or in the caption/takeaway).
- Never misname so badly the audience can't tell what tool is meant.
